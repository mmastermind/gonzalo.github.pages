    Summary Latitude vs X

    The purpose of this project was to analyze the changes how weather changes as you get close to the equator. To accomplish this analysis, we first pulled data from the OpenWeather API to assemble a dataset on over 500 cities.

    After assembling the dataset, Matplotlib was used to plot various aspects of the weather vs latitude. Factors we looked at included: temperature, cloudiness, wind speed and humidity. This site provides the source data and visualizations created as part fo the analysis, as well as the explanations and descriptions of any trends and correlations witnessed.
